# What

- The heart of the world tree

# Environment

- unity 2022.2.0f1
- scriptingBackend : IL2CPP
- apiCompatibilityLevel : .NetFramework

# How To Install

Add the lines below to `Packages/manifest.json`

- for version `1.2.14`

```csharp
"com.pancake.heart": "https://github.com/pancake-llc/heart.git#1.2.14",
```
